项目介绍
==================================================
页面布局种类有很多，但万变不离其宗，归纳总结出11中布局方式。
常见的Web页面布局分类汇总，欢迎补充！

示例 Demo
--------------------------------------------------
- [http://rochappy.github.io/layout](http://rochappy.github.io/layout)

开发方式
--------------------------------------------------
使用sass预编译处理css，需安装 [Node.js](http://nodejs.org) , npm , Grunt , ruby

安装依赖
```bash
cd lib && npm install
```

lib目录运行，编译 Sass
```bash
grunt sass
```

lib目录运行，实时监控编译 Sass
```bash
grunt watch
```

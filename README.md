##开发方式
基于 [nobone](http://github.com/ysmood/nobone) 开发
###安装 node_modules
```bash
npm install
```

###运行
```bash
nobone
```
默认port：`8013`

或运行
```bash
nobone -p 8017
```

##支持浏览器
- IE6+
- Chrome
- Firefox
- Opera
- Safari
